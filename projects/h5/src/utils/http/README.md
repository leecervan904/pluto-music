## Axios

### feature

+ 拦截器
+ 取消请求
+ 表单数据
+ 文件上传（大文件上传、断点续传、）
+ 请求池管理：请求队列、请求并发、失败重试

可选：
+ 错误优先返回
+ 请求参数、返回数据 Modal 推导


### process

+ 请求前
  1. 调用请求函数
  2. 请求参数处理
  3. 请求拦截器（主要为修改请求信息）
    1. 修改请求头部
    2. 修改用户标识（token 等）
+ 发起请求
+ 收到响应
  1. 响应拦截器（主要为错误处理）
    1. 网络错误处理
    2. 授权错误处理
    3. 普通错误处理
  2. 返回参数处理
